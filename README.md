# Art Space App

Jetpack Compose ile geliştirilmiş kişisel galeri uygulaması.

## Özellikler
- Görüntülerin gösterimi
- Fotoğraflar arasında gezinme (Next / Previous)
- Mobil ve tablet için uyumlu tasarım
- İki dilli arayüz (Türkçe ve İspanyolca)

# Art Space App

Aplicación de galería personal desarrollada con Jetpack Compose.

## Funcionalidades
- Visualización de imágenes
- Navegación entre fotos (Next / Previous)
- Diseño adaptable para móvil y tablet
- Interfaz bilingüe (turco y español)
